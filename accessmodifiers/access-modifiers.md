# Access Modifiers

## Types of Access Modifiers
1. Private
2. Public
3. Protected
4. Default (private-protected)  // Try to avoid using default

## General Convention
-  Try to keep your class `public`. Each file can only have one public class so keep classes separate.
- Try to keep your variables `private` and use `getter` and `setter` methods to access them.
- Use Protected when we want a method or variables to be accessed by sub-class, use `protected`.
