# Java Naming Convention
- For Class, Enum,Interface, Annotation name, use PascalCase eg: `MyClass`, `MyInterface`.
- For Method and Variable name, use camelCase eg: `myMethod`, `myVariable`.
- For Constant name, use UPPERCASE_SNAKE_CASE eg: `MY_CONSTANT`.
- For Package name, use lowercase eg: `com.example.myapp`.
- For Enum Constant name, use UPPERCASE_SNAKE_CASE eg: `MY_CONSTANT`.
- For Enum Field name, use camelCase eg: `myField`.
- For Enum Method name, use camelCase eg: `myMethod`.
