# CLASS and OBJECTS

- In Java Programming, a Class file is a **blueprint of an Object**.

- Java is an Object Oriented Programming(OOP) Language and everything is treated as an object. For example, humans, car, pen etc. all are treated as an object.

- Each object has Properties and Behaviour. For example, a **pen** has properties such as color, manufacturer name, type_of_pen. And behaviour such as write, erase etc.

- **new** keyword is used to create an object of a class. It allocates memory to the object in the machine.
