## Java Development Kit (JDK), Java Virtual Machine (JVM) and Java Runtime Environment (JRE)

- **JDK** -> converts program into byte code
- **JVM** -> Runs Byte code to execute the program
- **JRE** -> an environment where we can run the code. It contains JVM, utilities required to run Java codes.

- **JDK**(outer layer) -> **JRE**(middle layer) -> **JVM**(inner layer)
